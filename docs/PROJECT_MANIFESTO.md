# 📑 Project Manifesto: The Art of "Zero-Loading"

## 1. 🎯 The Core: 단 하나의 질문
> **"10만 명이 동시에 '로그인' 버튼을 누르는 그 1초, 서버는 응답할 수 있는가?"**

이 프로젝트의 시작은 기술적 호기심이 아닙니다. 티켓팅이나 선착순 이벤트에서 우리가 마주하는 '로그인 무한 로딩'이라는 절망적인 사용자 경험을 기술적으로 완전히 박멸하기 위함입니다. 우리의 코어는 **"트래픽 스파이크에서의 무중단 인증"**입니다.

---

## 2. 🔄 The Evolution: 시행착오와 진화 (The Journey)

프로젝트는 고정된 설계가 아니라, 한계를 마주하고 이를 깨부수는 사이클을 통해 진화해왔습니다.

### Cycle 1: 전통의 한계 (Phase 1)
*   **시도**: Java + Spring Security + Redis 세션 기반 인증.
*   **시행착오**: 가상의 1만 명 동시 접속 시나리오에서 Redis I/O가 병목이 됨을 확인. 서버가 늘어나도 '세션 저장소'라는 중앙 집중형 병목 때문에 로딩이 발생함.
*   **교훈**: "대용량 트래픽에서 '상태(State)'는 적이다."

### Cycle 2: 상태의 제거 (Phase 2 ~ 3)
*   **선택**: Stateless JWT 아키텍처로 전면 전환.
*   **이유**: 서버가 누구에게도 물어보지 않고(I/O 없이) 스스로 토큰을 검증하게 하여 응답 속도를 극대화함.
*   **시행착오**: 보안을 위해 도입한 RTR(Refresh Token Rotation)이 다시 Redis 조회를 유발함.
*   **돌파구**: Access Token 검증은 100% Stateless하게, 토큰 갱신(RTR)만 최소한의 I/O로 처리하는 '이중 구조' 확립.

### Cycle 3: 언어의 본질 (Phase 3.5)
*   **선택**: Java에서 Kotlin으로의 전면 전환.
*   **이유**: 대용량 트래픽 로직을 다룰 때, 자바의 장황한 코드는 '실제 성능 로직'을 가림. Kotlin의 Null-Safety와 간결함은 극한 상황에서의 안정성을 보장하는 '안전 장치'임.

---

## 3. 🎭 가상 시나리오 & 검증셋 (The Proof)

우리는 단순히 코드가 돌아가는 것을 확인하지 않습니다. 다음 가상 시나리오를 통과해야만 '성공'으로 간주합니다.

### [Scenario: The 10:00 AM Spike]
*   **상황**: 오전 10시 정각, 전국에서 10만 명의 유저가 동시에 '로그인' 버튼 클릭.
*   **검증셋 (Success Metrics)**:
    1.  **Latency**: 99%의 사용자가 100ms 이내에 토큰을 발급받아야 함.
    2.  **Scalability**: 서버 인스턴스를 2배로 늘렸을 때, 처리량(TPS)이 최소 1.8배 이상 선형적으로 증가해야 함.
    3.  **Resilience**: 인증 DB가 일시적으로 지연되어도, 이미 로그인된 유저의 서비스 이용은 0ms 지연으로 유지되어야 함.

---

## 4. 🌐 Expanding Core: 확장의 원리

우리는 코어에서 시작하여 다음의 원심력으로 프로젝트를 확대합니다.

1.  **Core (인증 엔진)**: Stateless JWT 발급 로직 최적화.
2.  **Inner Ring (보호막)**: Rate Limiting으로 매크로 차단, Distributed Lock으로 정합성 사수. (Phase 4)
3.  **Outer Ring (물리 엔진)**: Coroutine/R2DBC로 쓰레드 병목 제거, 가용 자원 극대화. (Phase 5)
4.  **Edge (완성)**: Native Image 빌드로 서버 확장 속도(Cold Start)를 0.1초대로 단축. (Phase 7)

---

## 5. 💡 Conclusion: 기술보다 '이유'
우리가 **Kotlin**을 쓰는 이유는 유행이 아니라 **안정성** 때문이고, **JWT**를 쓰는 이유는 편리함이 아니라 **확장성** 때문입니다. 이 프로젝트는 **"기술적 수단이 목표(로딩 0초)를 어떻게 현실로 만드는가"**를 증명하는 하나의 거대한 실험실입니다.
